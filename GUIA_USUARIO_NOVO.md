## 🔌 **Configure Seu Sistema Solar**

### **Passo 1: Vá para "Meu Sistema"**
- No menu lateral esquerdo
- Clique em **"Meu Sistema"**

### **Passo 2: Selecione a Marca do Seu Inversor**
Você verá **5 opções visuais** (cards):

| Marca | Quando Usar |
|-------|-------------|
| 🌐 **Growatt** | Se seu inversor é Growatt |
| 📡 **Solarman/Deye** | Se usa Deye ou Solarman |
| ⚡ **GoodWe** | Se seu inversor é GoodWe |
| 🏢 **Huawei** | Se usa Huawei FusionSolar |
| 🔧 **API Genérica** | Para outros fabricantes |

**Como descobrir sua marca:**
- Olhe na etiqueta do inversor
- Acesse o app/portail do fabricante
- Procure "Growatt", "Deye", "GoodWe", etc.

### **Passo 3: Clique na Sua Marca**
- O card ficará **verde** (selecionado)
- Aparecerá um formulário abaixo

### **Passo 4: Preencha os Dados**
```
📍 INFORMAÇÕES BÁSICAS
├─ Modelo do inversor: [selecione na lista]
├─ Potência instalada: [ex: 5000] W
├─ Localização: [São Paulo, SP]
└─ Distribuidora: [Energisa]

🔐 CREDENCIAIS DE ACESSO
├─ Método: [Login do Portal] ou [Token API]
│
└─ Se "Login do Portal":
   ├─ Usuário: [seu-email@fabricante.com]
   └─ Senha: [sua-senha-do-portal]
│
└─ Se "Token API":
   └─ Token: [cole-o-token-aqui]
```

### **Passo 5: Teste a Conexão**
- Clique em **"Testar conexão real"**
- Sistema conecta à API do fabricante
- Aguarde a resposta...

**Possíveis resultados:**
- ✅ **Verde:** "Inversor conectado com sucesso!"
- ⏳ **Amarelo:** "Validação pendente..."
- ❌ **Vermelho:** "Verifique as credenciais"

### **Passo 6: Salve a Integração**
- Se teste passou: clique **"Salvar integração"**
- Pronto! Sistema configurado.

## 🔑 **Como Obter Credenciais**

### **Para Growatt:**
1. Acesse: `https://server.growatt.com`
2. Crie conta ou faça login
3. Vá em "Configurações" → "API" ou "Developer"
4. Gere token ou use email/senha

### **Para Deye/Solarman:**
1. Baixe app "Solarman" ou "Deye"
2. Cadastre sua conta
3. Vá em configurações → API
4. Gere token de acesso

### **Para GoodWe:**
1. Acesse: `https://www.goodwe.com`
2. Login no portal
3. Vá em "API Management"
4. Gere credenciais

### **Para Huawei:**
1. Acesse: `https://eu5.fusionsolar.huawei.com`
2. Login com conta Huawei
3. Vá em "API Access"
4. Gere token

---

## ❓ **Problemas Comuns**

### **"Credenciais inválidas"**
- Verifique email/senha no portal do fabricante
- Certifique-se que a conta está ativa
- Tente gerar novo token

### **"Inversor offline"**
- Verifique se há internet no local
- Reinicie o inversor (30 segundos)
- Aguarde alguns minutos

### **"API não responde"**
- Verifique URL da API (preenchida automaticamente)
- Tente novamente em 5 minutos
- Contacte suporte se persistir
