## ❓ **PROBLEMAS COMUNS**

### **"Credenciais inválidas"**
```
🔍 VERIFIQUE:
├── Email/senha corretos no portal do fabricante
├── Conta ativa
├── Token não expirado
└── Tente gerar novo token
```

### **"Inversor offline"**
```
🔍 VERIFIQUE:
├── Internet funcionando no local
├── Inversor ligado (reiniciar 30s)
├── Aguardar 5 minutos
└── Contactar fabricante
```

### **"API não responde"**
```
🔍 VERIFIQUE:
├── URL da API correta
├── Fabricante com manutenção
├── Tentar novamente em 10 minutos
└── Suporte: suporte@sua-plataforma.com
```
